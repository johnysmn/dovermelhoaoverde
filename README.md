Do Vermelho ao Verde

Do Vermelho ao Verde é um guia prático de finanças pessoais criado para quem quer sair do aperto, organizar a vida financeira e começar a construir um futuro mais seguro, mesmo ganhando pouco.  Com uma linguagem simples, direta e sem “economês”, este ebook mostra passo a passo como entender seus gastos e controlar dívidas.
